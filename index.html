Question No-1

Difference between div and span

        Div stands for division. It means that div tag defines a division or section in HTML document.
        It is a block-level element which means it acquires whole width of the screen.
        So the next element will always come in next line. It can be used to style blocks of the content.
        Whereas Span tag is an inline element, it acquires only the width which is required for the
        content so the next element will be starting from the same line only.It can be used to style parts
        of the text.


Question No-2

Difference between class and IDs

        Class attribute is used to specify a class for an HTML element. Multiple elements can share the
        same class. It is often used to point to a class name in style sheet. For style an element, We
        use class="box" OR .box{} . One element can have more than one class.
        ID attribute is used to specify a unique id for an HTML element. Single element can share only
        one specific id. It is often to point to a style declaration in style sheet. For style an element,
        We use id="heading" OR #heading. One element can have only one id.

Question No-4

How many types of heading are there ? How to decide which to use where

        Heading tags are titles or subtitles that we want to display on our webpage. There are six levels
        of heading tags are there in HTML.
        h1 tag you should when you want to give the main heading of the page.
        h2 h3 tags you should use to give the sub-headings in the page.
        h4 h5 h6 tags you should use to give the content.
        As you move from h1 to h6 the importance of the heading decreases. h1 is of
        highest importance and h6 is of least importance.


Question No-5

Difference between block and inline

        An HTML element has a default display value depending on what type of element it is. There are
        two type of display values.
        Block elements acquire the entire width of the screen which means it always starts on a new line
        and browsers automatically add some space line and after the element.
        Examples:- div tag, p tag, table tag etc.
        Inline elements acquire only width required for the content which means it does not start on a
        new line.
        Examples:- anchor tag , span tag, image tag, input tag etc.

Question No-7

What are attributes?

        In HTML, attributes are special words which provide additional information about the
        elements and are the modifiers of HTML elements. Attributes should be written in lowercase
        only as they are case sensitive. It should always be applied with its name and value pair.
        It should always be applied with the start tag, we can add multiple attributes to an element
        by giving space between two attributes.
        Examples:- src , alt ,type ,value , id , name etc


Question No-8

Difference between HTML tags and elements

        HTML tags- Tags are the starting and ending part of an HTML element. They begin with the “<”
        symbol and end with “>”. Whatever written inside < and > are called tags. 
        Example:- div div is a html tag.
        HTML elements- Elements enclose the contents in between the tags. They consist of some kind
        of structure. It generally consists of a start tag, content and end tag.
        Example:- content is a html element.
